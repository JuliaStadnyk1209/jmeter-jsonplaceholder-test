# Висновки за результатами стрес-тесту

## Навантаження:

- 100 користувачів: API працює стабільно
- 200 користувачів: збільшується час відповіді
- 500 користувачів: частина запитів дає помилки (Response Timeout / 500)
- 1000 користувачів: API перестає стабільно відповідати

## Висновок:
Максимальна стабільна межа для JSONPlaceholder API становить **близько 200-300 одночасних користувачів** при стандартному використанні.
